# Con Cruise API

## Description

This implementaion uses the [hungarian algorithm](https://en.wikipedia.org/wiki/Hungarian_algorithm) to solve an optimization problem in demand fulfillment between customers and the cruisers.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
